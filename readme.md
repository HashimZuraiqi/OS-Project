
# Performance Evaluation of CPU Scheduling Algorithms

**Course:** CS11335 Operating Systems  
**Institution:** Princess Sumaya University for Technology  
**School:** King Hussein School of Computing Sciences  

---

## Project Overview

This project involves implementing and evaluating six CPU scheduling algorithms to compare their performance in terms of average turnaround time and average waiting time.
